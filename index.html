Code Sample: KPI Parser


import json
from statistics import fmean

# Load quarterly revenue from a JSON API response
with open('data/earnings.json') as f:
    obj = json.load(f)
revenue = [q['revenue_usd_m'] for q in obj['quarters']]
margin  = [q['op_margin_pct'] for q in obj['quarters']]

# Simple derived metrics
rev_growth = (revenue[-1] - revenue[-5]) / revenue[-5]
pc_margin  = fmean(margin)
print(f"TTM growth={rev_growth:.2%}, Avg margin={pc_margin:.1f}%")
Line highlighting draws attention to the key logic. Output is illustrative; in a live demo we could fetch data client-side.

Financial Math

Net Present Value (discrete cash flows):

\[ \mathrm{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} - C_0 \]

Dividend Discount (Gordon Growth):

\[ P_0 = \frac{D_1}{r-g} \quad \text{where } r>g \]
